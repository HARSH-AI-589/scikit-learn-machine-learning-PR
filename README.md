# Employee Performance Prediction using Scikit-learn

This project builds a Machine Learning model using the **Scikit-learn library** to predict employee performance based on multiple features such as age, experience, salary, work hours, and projects completed.

The goal of the project is to demonstrate a complete **machine learning workflow**, starting from data generation to model evaluation.

## Project Workflow

The project follows a structured machine learning pipeline:

1. **Dataset Creation**
   A synthetic dataset of 1000 employees was generated with logical relationships between variables.

2. **Data Preparation**

   * Feature selection
   * Train–test split
   * Data preprocessing

3. **Model Training**
   A machine learning classifier from the **Scikit-learn library** was trained on the dataset to learn patterns between employee attributes and performance.

4. **Model Evaluation**
   The trained model was evaluated using unseen test data to measure its prediction accuracy.

## Features Used

* Age
* Experience
* Salary
* Work Hours
* Projects Completed

## Target Variable

* **Performance**

  * 1 → High Performance
  * 0 → Low Performance

## Results

The model achieved approximately **95% prediction accuracy** on the test dataset, showing that it successfully learned the relationships in the data.

## Technologies Used

* NumPy
